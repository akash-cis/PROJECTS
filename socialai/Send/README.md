## ğŸ’« Deploy
The UI is hosted in S3 buckets and fronted by CloudFront distributions.

### Build
To build a distributable package run, for example for staging:

```
npm run build:stage
```

This will place the built files in the `public` folder

### Deploy
To deploy, copy the files in public up to the appropriate s3 bucket, for example to deploy to stage:

```
npm run deploy:stage
```

The files will now be up in the S3 bucket, since we're fronting these with CloudFront, it may be necessary to go to the CloudFront distribution in the AWS console and invalidate the cache before the updates will be visible to the world.
